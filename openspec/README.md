# OpenSpec Changes

OpenSpec changes define feature contracts, acceptance criteria, constraints,
and explicit exclusions. They are authoritative for feature behavior.

Create each change from the reusable [change templates](templates/change/).
The templates standardize document structure; accepted change documents define
the actual requirements and remain authoritative.
