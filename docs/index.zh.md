# ForgeFlow 文档系统

这是 ForgeFlow 长期维护工程知识的总入口。

## 从这里开始

- [产品](product/index.zh.md)：愿景和里程碑路线图。
- [架构](architecture/index.zh.md)：基础材料和评估。
- [工程流程](process/index.zh.md)：工作流、治理、生命周期、实现执行、Git 和审查规则。
- [里程碑](milestones/index.md)：实施计划、进度和 Phase Completion Record。
- [文档注册表](registry.zh.md)：每类文档的权威性、生命周期和更新规则。

## 权威文档面

- [RFC](../rfcs/README.md)：架构提案和长期方向。
- [ADR](../adr/README.md)：已接受的架构决策。
- [OpenSpec](../openspec/README.md)：功能契约和验收标准。
- [复盘](../retrospectives/README.md)：已完成里程碑的学习记录。

历史且非权威的材料位于 [_history](_history/README.md)。
