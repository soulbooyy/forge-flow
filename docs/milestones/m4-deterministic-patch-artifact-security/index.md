# Milestone 4: Deterministic Patch Artifact and Security Scanning

## Scope

Build metadata-only, immutable PatchIntent/PatchArtifact and security facts
from M2 PatchProposal lineage. Excludes source access, raw diff/patch material,
patch application, execution, approval, persistence, DurableRunSummary, and
GitHub effects.
