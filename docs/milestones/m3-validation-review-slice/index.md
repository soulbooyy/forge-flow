# Milestone 3: Validation and Review Slice

## Scope

M3 establishes fixture-only, immutable `ValidationResult`,
`ValidationTerminal`, and `ReviewResult` contracts with policy, artifact, and
evidence lineage. It excludes real command execution, sandbox/runtime
integration, workspace access, retry, repair, and all external side effects.
